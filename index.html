BLE flow
1App opens Bluetooth picker — select "LumaRoK-LMR-XXXXXX"
2Credentials sent directly over BLE
3ESP32 saves to memory and reboots
4Unit connects to WiFi then HiveMQ

MQTT fallback — only works if the ESP32 is already online. Use BLE for first-time setup.
